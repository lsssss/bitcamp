## ver 04
- 배열 문법 응용

여러 명의 성적을 저장하고 다음과 같이 출력하라!

출력내용:
```
홍길동, 100,  90,  80, 270,  90.0
임꺽정,  80,  80,  80, 240,  80.0
유관순, 100, 100, 100, 300, 100.0
```
